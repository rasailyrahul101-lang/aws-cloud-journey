# Day 7 — SDLC 🔄

## What is SDLC?
Software Development Life Cycle — a structured process for
building and maintaining software products.

## The 7 Phases

| Phase | What Happens |
|---|---|
| **Planning** | Client + team discuss purpose, end users, requirements |
| **Requirement Analysis** | Detailed requirements gathered and documented |
| **Design** | Architecture, UI, database structure designed |
| **Implementation** | Developers write code using languages and tools |
| **Testing** | QA team tests in multiple environments, bugs sent back to devs |
| **Deployment** | Software released to users, installation links set up |
| **Maintenance** | Bug fixes, updates, performance improvements |

## Where Does a Cloud Engineer Fit?

| Phase | Cloud Engineer Role |
|---|---|
| **Planning** | Estimate cloud costs, choose AWS services |
| **Design** | Design cloud architecture — VPC, EC2, RDS, S3 |
| **Development** | Provide infrastructure for developers to build on |
| **Testing** | Set up test environments on AWS |
| **Deployment** | Deploy using CI/CD pipelines |
| **Maintenance** | Monitor with CloudWatch, scale, fix incidents |

## Key Insight
> Developer builds the car 🚗
> Cloud Engineer builds the road and fuel stations 🛣️
