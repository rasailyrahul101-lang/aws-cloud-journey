# Day 5 — Git & GitHub Notes

## Core Commands
| Command | What it does |
|---|---|
| `git init` | Initialize empty repo |
| `git add` | Stage changes |
| `git commit` | Save snapshot |
| `git push` | Upload to GitHub |
| `git pull` | Download from GitHub |
| `git clone` | Copy a repo |
| `git branch` | Create branch |
| `git checkout` | Switch branch |
| `git merge` | Combine branches |
| `git rebase` | Linear history |
| `git log` | Commit History |

## Merge vs Rebase
- Merge — keeps full history, shows branching (Used in team)
- Rebase — clean linear history, copies commits (Used Alone)
